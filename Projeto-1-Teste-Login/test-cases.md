# 📋 Casos de Teste – Login

### ✅ CT01 – Login com dados válidos
**Passos:**
1. Acessar tela de login  
2. Inserir email válido  
3. Inserir senha válida  
4. Clicar em entrar  

**Resultado esperado:**
Usuário deve acessar o sistema com sucesso  

---

### ❌ CT02 – Login com senha inválida
**Passos:**
1. Inserir email válido  
2. Inserir senha incorreta  

**Resultado esperado:**
Sistema deve exibir mensagem de erro  

---

### ⚠️ CT03 – Campos vazios
**Passos:**
1. Clicar em entrar sem preencher  

**Resultado esperado:**
Sistema deve impedir login e alertar usuário
